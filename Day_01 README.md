Class:
        A class is a blueprint or template used to create objects.
Object:
        An object is a real instance of a class.



Access Modifiers:
                  Access modifiers are keywords in Java that define the visibility  of classes, methods, variables,  constructors.

They are of four types:

            1. public:
                      Accessible from any class, any package, and any subclass.
            2. private:
                       Accessible only within the same class
            3. protected:
                      Accessible in same class,same package and same subclasses .
            4. Default :
                      Accessible in same class and same package.
